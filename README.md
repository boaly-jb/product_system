# product_system
This webpage allows users to add and manage products easily. Each product can include details such as name, price, image, and category. Products are stored in a database and displayed dynamically. Users can filter products by category, making it simple to browse and find items quickly in an organized way.


Description:

This webpage is designed to manage and display products in an organized way. Users can easily add new products with details such as product name, price, image, and category. All products are stored in a database and displayed dynamically on the webpage.

The page also includes a category-based filtering system that allows users to quickly find products according to their selected category. When a category is selected, only the products belonging to that category will be shown, making the browsing experience faster and more user-friendly.

This system helps maintain a structured product catalog, improves product management, and allows users to explore products efficiently.

Key Features:

Add new products with details (name, price, image, category)

Store product data in a database

Display products dynamically on the webpage

Filter products based on categories

User-friendly and organized product listing
